## Description

NWA Coders - Code Snippets

## Installation

```bash
$ npm install
```

```bash
$ cp .env.example .env
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app dev tools

[Local API Document Endpoints](http://localhost:3000/api).

Generate a resource 

```bash
$ nest g resource
```
