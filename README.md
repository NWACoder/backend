## Description

NWA Coders - Code Snippets

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app dev tools

[API Document Endpoints](http://localhost:3000/api).

Generate a resource 

```bash
$ nest g resource
```
